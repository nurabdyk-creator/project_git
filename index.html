Welcome to GIT lesson

This is the Git lesson for batch 16..

Step by step guide:

  1. Install Git from https://git-scm.com/install/
  2. Configure Git with your name: git config --global user.name "Your Name"
  3. Configure Git with your name and email: git config --global user.email "your.email@example.com"
  4. Initialize git code: git init
  5. Merge commits: git merge branch-name
